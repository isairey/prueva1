# ☕ Spring Boot - Calculadora de Promedio

Proyecto desarrollado en **Spring Boot** para calcular el promedio de un conjunto de números.  
Incluye una implementación inicial con errores y una versión corregida para fines de aprendizaje y pruebas.

---

## 🚀 Descripción

Este proyecto permite calcular el promedio de una lista de valores numéricos a través de una lógica simple implementada en Java.  
Es ideal para comprender:

- Manejo de colecciones en Java
- Lógica de cálculo básico
- Estructura de un proyecto con Spring Boot
- Corrección de errores en código
